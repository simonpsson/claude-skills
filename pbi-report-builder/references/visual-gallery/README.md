## Adding Screenshots

1. Take a screenshot of your visual in Power BI Desktop
2. Save as PNG in `images/` subfolder
3. Name format: `{visualType}-{variant}.png` (e.g., `cardVisual-kpi-yoy.png`)
4. Update this table with the new entry
